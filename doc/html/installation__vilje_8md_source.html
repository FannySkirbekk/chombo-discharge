1 Compilation on vilje.hpc.ntnu.no {#installation-vilje}
2 ================================
3 
4 TLDR
5 ----
6 Load modules by
7 
8  module load intelcomp/15.0.1
9  module load gcc/4.9.1
10  module load mpt/2.10
11 
12 The configuration file for vilje is found here. Store it as chombo_root/lib/mk/Make.defs.local
13 
14 Loading the modules
15 -------------------
16 To compile the system on the *vilje* supercomputer, you need to load compilers and libraries before compiling the code. We recommend that you use intel compilers by loading
17 
18  module load intelcomp/15.0.1
19 
20 which will load Intel compilers for Fortran and C/C++ (version 15.0.1). An updated version of gcc is loaded by
21 
22  module load gcc/4.9.1
23 
24 MPI libraries are loaded by
25 
26  module load mpt/2.10
27 
28 and HDF5 is loaded by
29 
30  module load hdf5/1.8.14
31 
32 Modifying the configuration file
33 --------------------------------
34 
35 It is likely possible to load newer versions of compilers, MPI, and HDF5 (but we haven't tried). Once the libraries are loaded, the code may be compiled as usual. It will be necessary to update CHOMBO_HOME and supply the library path to the Chombo makefiles.
36 CHOMBO_HOME is simply updated to the folder in which Chombo is installed. HDF5 paths are found by typing
37 
38  module disp hdf5/1.8.14
39 
40 which will display the library paths to HDF5. The typical output is
41 
42  /sw/sdev/modulefiles/hdf5/1.8.14:
43 
44  module-whatis general purpose library and file format for storing scientific data
45  prereq intelcomp/15.0.1
46  prereq mpt/2.10
47  prepend-path PATH /sw/sdev/Modules/hdf5/hdf5-1.8.14/bin
48  prepend-path CPATH /sw/sdev/Modules/hdf5/hdf5-1.8.14/include
49  prepend-path FPATH /sw/sdev/Modules/hdf5/hdf5-1.8.14/include
50  prepend-path LIBRARY_PATH /sw/sdev/Modules/hdf5/hdf5-1.8.14/lib
51  prepend-path LD_LIBRARY_PATH /sw/sdev/Modules/hdf5/hdf5-1.8.14/lib
52  prepend-path PYTHONPATH /sw/sdev/Modules/hdf5/hdf5-1.8.14/lib/python2.7/site-packages
53  setenv HDF5_DIR /sw/sdev/Modules/hdf5/hdf5-1.8.14
54 
55  Loads HDF5 version 1.8.14
56 
57 To use Intel's Fortran and C/C++ compilers, please specify
58 
59  FC = ifort
60  CXX = icpc
61 
62 in Make.defs.local. Likewise, update Make.defs.local so that it finds the HDF5 libraries and include files. For this version of HDF5, they are located in
63 
64  /sw/sdev/Modules/hdf5/hdf5-1.8.14/include
65  /sw/sdev/Modules/hdf5/hdf5-1.8.14/lib
66 
67 With these paths, the appropriate flags in Make.defs.local are
68 
69  HDFINCFLAGS = -I/sw/sdev/Modules/hdf5/hdf5-1.8.14/include
70  HDFLIBFLAGS = -L/sw/sdev/Modules/hdf5/hdf5-1.8.14/lib -lhdf5 -lz
71 
72  HDFMPIINCFLAGS = -I/sw/sdev/Modules/hdf5/hdf5-1.8.14/include
73  HDFMPILIBFLAGS = -L/sw/sdev/Modules/hdf5/hdf5-1.8.14/lib -lhdf5 -lz
