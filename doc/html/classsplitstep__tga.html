Detailed Description

Class for evolving plasma equations with a split-step RK2/TGA method.

This class uses RK2 advancements for the advection and source terms, and an implicit TGA method for the diffusion part.

Class options

splitstep_tga.rk2_alpha = 1.0 # Set alpha. 1.0 = Heuns method, 0.5 = Midpoint method splitstep_tga.turn_off_advection = false # Turn off advection, debugging only splitstep_tga.turn_off_diffusion = false # Turn off diffusion, debugging only splitstep_tga.turn_off_source = false # Turn off source term, debugging only splitstep_tga.turn_off_poisson = false # Turn off Poisson, debugging only splitstep_tga.turn_off_rte = false # Turn off RTE, debugging only
