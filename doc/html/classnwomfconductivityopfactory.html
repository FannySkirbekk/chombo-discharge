Detailed Description

Factory class for generationg nwomfconductivityop's.

The jump condition is of the Neumann type in the form

\[ a\partial_n\phi_1 - b\partial_n\phi_2 = c\sigma, \]

where \(a\), \(b\) and \(c\) are variables.

Note
This class generates its own EBConductivityOps as well as it's own interpolation utilities.

This class only supports two-phase solutions with single components
